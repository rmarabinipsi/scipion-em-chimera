Command: scipionshellcrown

Usage:
scipionshellcrown modelId radius outModelId orientation XX sphereFactor YY crownwidth ZZ

The command scipionshellcrown creates a crown from a 3D Map using a a quasy-spherical shell.

where

  • modelId: model id of the 3D map to be analyzed
  • radius: radius at which the density is going to be shown (Angstroms)
  • outModelId: id of the output icosahedron showing the density surface
  • orientation: symmetry used to create the icosahedron used to define the surface to be shown (default 222r)
  • sphereFactor: weight of the sphere component in an interpolation between an icosahedron and a sphere of equal radius (1.0: sphere; 0.0: icosahedron) (default=1)
  • crownwidth: shell width (default = 10A, the crown goes from the selected radius - crownwidth/2 to selected radius + crownwidth/2)

Examples

scipionshellcrown #1 250 10 orientation 222r sphereFactor 0.7 crownwidth 5
will create a3D map with a shell pf thickness 5A of the model #1. The outher shell is defined by spherical icosahedron with symmetry 222r and sphereFactor 0.7. The model id of the output will be 10.

Remark

  • In case the 3D map and the icosahedron do not fit, select center in Origin index of Map Coordinates menu.
  • Although orthoplanes is the visualization option by default, you can select plane instead in the Volume Viewer menu to see only the transversal plane.
  • The plane/orthoplanes can be moved using the right mouse button
